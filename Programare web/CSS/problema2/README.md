Structura unui site ce contine: un header, un meniu (lista de elemente), un div principal cu continut si un footer.

Structura site-ului este stilizata folosind doua fisere CSS externe (referite pe rand in antetul paginii). Pentru unul din fisierele CSS meniul va fi orizontal, iar pentru celalalt vertical. Singurul lucrul permis a se modifica intre cele doua variante de vizualizare a site-ului este numele fisierului CSS la care se face referire in antetul paginii. Nu se vor face modificari la structura documentului.
